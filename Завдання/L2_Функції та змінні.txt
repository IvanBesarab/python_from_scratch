Задача 0. Вивести на екран найбільше з двох чисел. 
Підказка як згенерувати 2 випадкових числа в Python
Можна скористатися модулем random:

import random

a = random.randint(1, 100)  # випадкове число від 1 до 100
b = random.randint(1, 100)

print(a, b)

Для порівняння чисел використовують такі оператори:
* == — дорівнює
* != — не дорівнює
* > — більше
* < — менше
* >= — більше або дорівнює
* <= — менше або дорівнює
